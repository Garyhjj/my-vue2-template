## 使用的技术栈

- Vue 一个 js 库
- vuex 数据流管理 状态管理
- Sass 用编程的方式绘制样式
- Element 成熟的开源 UI 框架
- Axios 灵活的无状态数据请求
- webpack npm 包管理 资源配置

### 安装依赖
```
yarn
```

### 开发运行
```
npm run dev
```

### 生产打包
```
npm run build-prod
```

### 语法格式修改
```
yarn lint
```
